## To Setup the project:

### Nextjs:

```bash
cd client/web
pnpm i
pnpm dev
```

### Expressjs

```
cd server/rest
pnpm i
pnpm dev
```

Make sure to setup the .env files for both server and client and run the /init endpoint to hydrate the collection with laundries and an admin account

#### Server ENV

PORT=5000
MONGO_URI
SECRET
ADMIN_NAME
ADMIN_EMAIL
ADMIN_PASS
CLIENT_URL

#### Client ENV

NEXT_PUBLIC_SERVER_URL
